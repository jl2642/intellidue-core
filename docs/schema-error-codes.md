- `DECISION_READY_OPEN_GATES`
- `DECISION_READY_RESTRICTED_OUTPUTS`
- `FINAL_DD_OPEN_GATES`

- `MANIFEST_PATH_ORDER`
- `PACKAGE_CORRUPT`
- `PACKAGE_NOT_SINGLE_ROOT`
- `PACKAGE_UNSAFE_PATH`
- `PACKAGE_INVALID_ZIP`

Codes are machine-readable and stable within schema contract version `1.0.0`.
